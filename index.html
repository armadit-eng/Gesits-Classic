⚙️ Pengaturan

🗺️ Sumber Peta
🔑 Mapbox Access Token
🔑 MapTiler API Key
💡 Daftar gratis di maptiler.com → API Keys → create key. Free tier 100K tiles/bulan — lebih dari cukup untuk pemakaian harian.
🔋 Estimasi Energi & Jarak
💡 Kapasitas Total (Wh) = Voltase Nominal × Kapasitas (Ah). Sisa Energi = SOC% × Kapasitas Total. Contoh: 72V × 20Ah = 1440 Wh saat SOC 100%. Estimasi jarak otomatis memakai efisiensi real-time trip (Wh/km aktual) bila jarak trip sudah >0.3 km, jika belum memakai nilai referensi di atas.
⚡ OpenChargeMap (Database SPKLU)
💡 Database SPKLU jauh lebih lengkap dibanding sebelumnya. Daftar gratis di openchargemap.org/site/develop/api → generate API key → tempel di sini. Tanpa key tetap bisa jalan (kuota publik terbatas), dengan key lebih stabil & banyak request.
🗺️ Resolusi Snapshot AT+MAPS
Preset
💡 Ukuran lebih besar = gambar lebih detail tapi kualitas JPEG otomatis diturunkan supaya tetap muat di bawah 7000 karakter (batas data AT+MAPS). Layar motor kecil → 350×220 biasanya sudah cukup tajam; layar lebih besar bisa naikkan ukuran.
📟 Tampilan Layar Dashboard (AT+NAV)
💡 Data yang di-toggle aktif akan ditampilkan bergantian di layar motor lewat AT+NAV, sesuai interval di bawah. Saat navigasi aktif, nama jalan ikut masuk ke giliran yang sama.
